This is very simple example of jsreport embedding function.

You can find jsreport template for this example here

You can find sample excel sheet with correct structure here

This is the html5 code anyone can insert into his website

<div id="jsr-root" data-shortid="g1R9YLmgj" data-version="3"
data-fileInput="true" data-dataArea="true" />

<script>(function (d, s, id) {
var js, fjs = d.getElementsByTagName(s)[0];
if (d.getElementById(id)) return;
js = d.createElement(s); js.id = id;
js.src = "//jsreport.net/js/embed.js";
fjs.parentNode.insertBefore(js, fjs);
}(document, 'script', 'jsr-embeded'));
</script>
