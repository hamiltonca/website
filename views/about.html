jsreport about

Open source platform for designing and rendering various business reports.

It uses different approach than the most of the other commercial platforms. It's intention is not to provide an ultimate fancy UI with report designer like other does. Idea is that any fancy WYSIWYG editor will never be good enough to the developers. jsreport lets developers to define reports using javascript templating engines (like jsrender or handlebars).
