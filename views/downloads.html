We are currently working on the multiplatform seamless jsreport installer. Please try first our jsreport online service for free and we will prepare installer for you in the next couple of weeks.

If you are eager to try to install on your own server, please do the following steps. Don't hasitate to tweet me in case of problems @jan_blaha

  1. download and install latest nodejs
  2. download and install latest mongodb and start it
  3. cmd in your working directory npm install jsreport
  4. cmd in your working directory cd node_modules/jsreport
  5. cmd in your working directory node server.js

Now you should have your local jsreport server running. You can try to access https://localhost in your browser and it should display jsreport html studio.
