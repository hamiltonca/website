downloads

Windows
Installer
Download and run x64 windows installer jsreport_x64.exe to host jsreport in windows service.
Manual installation steps
  1. download and install latest node.js
  2. run cmd with admin rights
  3. cd to working directory
  4. type: npm install jsreport
  5. type: node node_modules/jsreport --init
  6. to start server type: node server.js
  7. to install windows service type: node server.js --install
IIS
  1. download and install latest node.js
  2. run cmd with admin rights
  3. cd to working directory
  4. type: npm install jsreport
  5. type: node node_modules/jsreport --init
  6. remove httpsPort specification from created prod.config.json
  7. download and place sample web.config into jsreport folder
  8. install iisnode Don't forget to have iis installed with all Application development sub features and also iis url rewrite extension.
  9. create standard iis website for jsreport folder and choose port and bindings you want, also don't forget to add security permissions to jsreport folder for IIS App Pool user when appropriate. IMPORTANT: Create a separate website. Don't just add application to the default website.
  10. try access website url in browser, jsreport should pop up
Unix
Installation steps
  1. install latest node.js particular for your linux distribution.
  2. go to console, navigate to your working directory
  3. type sudo npm install jsreport
  4. type sudo node node_modules/jsreport --init
  5. type sudo node server.js
  6. open browser and see https://localhost

You may find troubleshooting guide here.

Docker

You can use docker image jsreport/jsreport for easy installation

See jsreport docker repository for details.

Microsoft Azure

jsreport doesn't work on Azure Website due to limitations for printing pdf.

You need to set up Azure VM or Azure cloud service where both Windows and Linux images works well with jsreport. There is no particular configuration needed.

OpenShift

For OpenShift you need to configure host and port ranges for jsreport internal web servers. You can find how to do it in OpenShift installation instructions.
