Linux

We are currently preparing linux packages. Before it's finished, please follow these steps

  1. install latest node.js particular for your linux distribution.
  2. go to console, navigate to your working directory
  3. type sudo npm install jsreport
  4. type sudo node node_modules/jsreport --init
  5. type sudo node server.js
  6. open browser and see https://localhost

Use init script to install jsreport server pernamently.

You can use following for ubuntu

/etc/init/jsreport.conf

start on startup
chdir [working directory]
exec sudo node server.js

Don't forget to add execution rights with chmod -x jsreport.conf
