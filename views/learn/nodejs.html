node.js

Whole jsreport is written in node.js and its obvious it has first class support for node environment. It is distributed in a common way through npm package so it's easy to reach it.

It can run as a standalone express server in form of jsreport on premise or it can run as a part of another nodejs application. This documentation covers running jsreport as a part of another application in section adapting jsreport. Section node.js client covers connecting into running jsreport server remotely using dedicated node.js library.
