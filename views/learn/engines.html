Templating engines

jsreport uses javascript templating engines to define report layout. You can bind input data, use loops, conditions or javascript helpers... Templating engines basically provides you way to define any custom report you like in a very fast, flexible, matured and well known way for everyone who has ever written a javascript single page application.

jsreport ships out of the box with following engines.

handlebars
jsrender
EJS
pug
Custom Engine
