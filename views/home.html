Open Source

This product is being developed and maintained by the open-source community in which you can participate. Whole source code base of jsreport is hosted and publicly available on github. Pull-requests as well as proposals to improvments are more than welcome.

Extensible

Whole platform is assembled from separate pieces (extensions) which user can activate/deactivate. This extensions could be created by developers to extend jsreport functionality including studio user interface.

Quick to learn

You don't need to learn anything new when using jsreport. Everything is based on well known javascript templatinng engines like jsrender or handlebars.

Html Studio

jsreport contains extensible html-based management studio ouf of the box. Studio includes report designer as well as functionalitys to manage report templates and others.

Output formats

jsreport is great for rendering output formats based on xml like html. It can also easily convert html into pdf using phantomjs. Or even generate precise pdf using Apache FOP. Output formats can be easily extended by jsReport recipes.

REST API

jsreport provides whole functionality throught REST API. Since the most of the API is provided in the form of OData endpoint common frameworks from various languages could be used to consume it. See more
