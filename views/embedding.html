embedding

Every one can embed jsreport for free into his website and allow internet audience to use his jsreport template and render documents. Embedding allows to filter out the complexity of jsreport into just simple html form, where user uploads or fill input data, hit download and gets back document.

Native format for input data is json, but to make it easy to use for normal people, embeded version is able to internally convert text files or even excel files into json.

Usage is very simple, just hit the embed button in the playground template toolbar and dialog with proper informations will popup.
